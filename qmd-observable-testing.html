Observable + Quarto - 3 ways

1. robservable package:

The robservable R package “allows the use of Observable notebooks (or parts of them) as htmlwidgets in R.”1

library(robservable)
robservable(
  "@observablehq/test-observable-quarto",
  include = c("staticChart")
)

2. import:

Within {ojs} cells in a .qmd, we can use imports to access named cells from an Observable notebook:

import {staticChart} from "@observablehq/testing-robservable"
staticChart

3. {ojs} cells:

Or, we can write code directly in {ojs} cells:

testArray = [{name: "Teddy", weight: 50},
             {name: "Khora", weight: 60},
             {name: "Waffle", weight: 35},
             {name: "Banjo", weight: 40},
             {name: "Tallie", weight: 55}]
Plot.plot({
  marks: [
    Plot.barY(testArray, {x: "name", y: "weight", sort: {x: "y", reverse: true}}),
    Plot.ruleY([0])
  ]
})
